subroutine sdgvm1::set_landuse ( real(dp), dimension(max_cohorts)  ftprop,
integer  ilanduse,
real(dp), dimension(12,31)  tmp,
real(dp), dimension(12,31)  prc,
integer, dimension(8)  nat_map,
integer  nft,
real(dp), dimension(max_cohorts,max_years)  cluse,
integer  year,
integer  yr0 
)

Set lanuse by allocating the cover of new ground.

Assign gound available for new growth, through disturbance and mortality, given by ftprop. The values of the desired cover are given in the array 'cluse'. Note that any changes in the desired are only met if new ground is large enough to make up the difference. Over time the cover will reach any desired value and once reached the cover will remain constant.
